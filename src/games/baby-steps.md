A game about climbing a mountain without any help from anyone.

I worked on this full time between 2019 and 2025. I did a lot of kinds of work for it, systems design, level design, cinematography, acting, editing, improv. It will be hard to find another project that's so fulfilling on so many fronts, and is such a good vessel for so many ideas.
